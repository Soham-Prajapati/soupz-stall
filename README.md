# SOUPZ-STALL

soupz-agents — tools, docs, and agents for the SOUPZ-STALL project.

This repository contains node tooling, documentation, and persona agents for Soupz. See the docs/ directory and the included guides for setup and contribution notes.

## Quickstart

1. Install dependencies
   npm install

2. Run tests
   npm test

3. Run the project
   npm run start
