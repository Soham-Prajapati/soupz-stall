## 🎯 What is Soupz-Stall?

Soupz-Stall is a multi-agent AI orchestrator that gives you access to 24 expert personas, each with the ability to:
- Spawn subagents for parallel work
- Ask for user input interactively
- Hand off to other personas dynamically
- Collaborate like a real team

**Cost**: $0.00 (uses FREE models)

---

## 🌟 Key Features

### 1. 24 Expert Personas
From Architect to Designer to DevOps - every role covered

### 2. Subagent System
Every persona can spawn others for parallel execution

### 3. Interactive Mode
Personas can ask for your input and guidance

### 4. Dynamic Switching
Seamless handoffs between personas

### 5. Master Orchestrator
Handles complex 50+ line prompts with full team coordination

## 💡 Quick Examples

### Simple Usage
```bash
$ soupz-stall
> @architect design a REST API
```

### With Subagents
```bash
> @brainstorm generate ideas
# Spawns @evaluator to score ideas
```

### Master Orchestration
```bash
> @master [paste 50-line problem statement]
# Spawns 5 personas → waits → spawns 5 more
```
