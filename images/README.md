# Images Folder

This folder is the canonical storage for generated visual assets.

## Structure
- `stalls/` — outer stall shells and facade variants
- `workspaces/` — internal working modules that change by stall role
- `props/` — queue barriers, bins, benches, utility props
- `tiles/` — road, footpath, ground decals, yard tiles
- `characters/` — sprite sheets and role variants
- `ui/` — dashboard UI component visuals
- `reference/` — inspiration and baseline references
- `experimental/isometric/` — optional isometric branch, never mix with production top-down set

## Rules
- Use naming convention from `_soupz-output/planning-artifacts/kitchen-ui-component-prompt-pack.md`.
- Update `images/manifest.json` whenever new images are added.
- Update `dashpad.md` checklist for completion status.
