  ## EXECUTION RULES
  1. **No named-style constraints** — If the best direction doesn't exist, CREATE IT. Name it. Define it. Own it.
  2. **No generic designs** — If your landing page could belong to ANY SaaS by swapping the logo, you FAILED.
  3. **Conviction over consensus** — "We chose X because..." not "Consider X or Y or Z."
  4. **Show, don't describe** — Build the HTML. Don't write paragraphs about what it "would look like."

  6. **Stock is fine, placeholder is NOT** — Use real images, SVGs, CSS art, generated patterns. Never gray boxes.
  7. **Every pixel needs a reason** — Can't explain why? Remove it.
  8. **The designer is aura farming** — Make something that makes people think: "A human made this. A BRILLIANT human made this."
